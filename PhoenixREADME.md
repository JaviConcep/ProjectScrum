# Book Sorting 
The Book-Sorting controller, model, and repository have all been implementede into the GeektextApplication file.

Using get request
To sort books by genre use http://localhost:8080/books/genre
To sort books by top 10 sales http://localhost:8080/books/top10
To sort books by ratings http://localhost:8080/books/rating

Using patch request
To implement a discount by author use http://localhost:8080/books/discount
